Proyectos

Automatización de carga de campañas

Problema: Procesos manuales para cargar información desde Excel.

Solución: ETL en Python para limpiar, validar y cargar datos hacia MySQL.

Tecnologías: Python, pandas, MySQL, openpyxl.

Resultado: Mayor consistencia de datos y reducción de trabajo manual.

Dashboard de call center

Problema: Falta de visibilidad sobre productividad y operación.

Solución: Diseño de KPIs y dashboard ejecutivo en Power BI.

Tecnologías: SQL Server, Power BI, DAX.

Resultado: Mejor análisis de tiempos, volumen y desempeño operativo.

Integración de datos con SSIS

Problema: Información dispersa en múltiples fuentes.

Solución: Procesos ETL en SSIS para construir un Data Mart.

Tecnologías: SSIS, SQL Server, Power BI.

Resultado: Datos centralizados y listos para análisis de negocio.
